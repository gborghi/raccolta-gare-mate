# British Mathematical Olympiad 1979-80 — Round 2 — Quesito 3

*Prove equality condition for a sum identity over real numbers*

> Prove that if $n \geq 1$, $a_1, a_2, \ldots, a_n$ are real numbers, then
> $$\sum_{i=1}^{n} s_i(a_{i+1} - a_i) = (q-1)\sum_{i=1}^{n}(a_i - a_{i-1})^2,$$
> equality holding if and only if $a_i = i\,a_1$ $(q \leq i \leq n)$.
