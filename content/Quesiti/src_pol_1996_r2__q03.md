*Inequality with a+b+c=1 and a,b,c > -1/2*

> Prove that if $a, b, c > -\frac{1}{2}$ and $a + b + c = 1$, then
> $$\frac{a}{a^2+1} + \frac{b}{b^2+1} + \frac{c}{c^2+1} \le \frac{9}{10}.$$
