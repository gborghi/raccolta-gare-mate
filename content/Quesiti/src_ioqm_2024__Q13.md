*Find value of a given two equations in positive integers a,b,c with a>c*

> Three positive integers $a, b, c$ with $a > c$ satisfy the following equations:
> $$ac + b + c = bc + a + 66, \quad a + b + c = 32.$$
> Find the value of $a$.
