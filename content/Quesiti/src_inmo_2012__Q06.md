*Functional equation on integers: find all f values and set where f(n)≠0*

> Let $f : \mathbb{Z} \to \mathbb{Z}$ be a function satisfying $f(0) \neq 0$, $f(1) = 0$ and
> \begin{itemize}
> \item[(i)] $f(xy) + f(x)f(y) = f(x) + f(y)$;
> \item[(ii)] $(f(x-y) - f(0))f(x)f(y) = 0$,
> \end{itemize}
> for all $x, y \in \mathbb{Z}$, simultaneously.
> \begin{enumerate}
> \item[(a)] Find the set of all possible values of the function $f$.
> \item[(b)] If $f(0) \neq 0$ and $f(2) = 0$, find the set of all integers $n$ such that $f(n) \neq 0$.
> \end{enumerate}
