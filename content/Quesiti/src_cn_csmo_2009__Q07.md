*Find max and min of a rational function on the unit simplex*

> Let $f(x, y, z) = \frac{x(2x - y)}{1 + x + 3y} + \frac{y(2y - z)}{1 - y + 3z} + \frac{z(2z - x)}{1 + z + 3x}$, where $x, y, z \ge 0$ and $x + y + z = 1$. Find the maximum value and the minimum value of $f(x, y, z)$. (Posed by Liu Shenglong)
