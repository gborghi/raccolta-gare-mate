*72, 71 e 72^71 sono egiziani?*

> Un intero n si definisce egiziano se esiste una sequenza di interi positivi strettamente crescente 0 < a1 < a2 < ... < ak = n (quindi l’ultimo termine della sequenza è n) tale
> che
> 1
> a1
> + 1
> a2
> + · · · + 1
> ak
> = 1.
> (a) Determinare se il numero n = 72 è egiziano.
> (b) Determinare se il numero n = 71 è egiziano.
> (c) Determinare se il numero n = 7271 è egiziano.
