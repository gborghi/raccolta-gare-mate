# Kangourou 2004 - Student (marzo) — Quesito 19

*Segno prodotto in successione geometrica a3<a2<a4*

> In una successione geometrica  a 1, a 2, a 3, ... si hanno le seguenti disuguaglianze: a 3 <  a 2 <  a 4 .
> Allora sicuramente
> A) a 3 Aa 4 > 0                B)a 2 Aa 3 < 0   
> C) a 2 Aa 4 < 0
> D)a 2 < 0   
> E) a 2 Aa 3 > 0
