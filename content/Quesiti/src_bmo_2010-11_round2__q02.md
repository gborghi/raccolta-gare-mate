*Count positive integers n < 2011 with f(n) = f(2011)*

> The function $f$ is defined on the positive integers as follows:
> $$f(2n) = n \quad \text{if } n \text{ is even,}$$
> $$f(2n) = 2n \quad \text{if } n \text{ is odd,}$$
> $$f(2n+1) = f(n) \quad \text{if } n \text{ is even,}$$
> $$f(2n+1) = f(n)+1 \quad \text{if } n \text{ is odd.}$$
> Find the number of positive integers $n$ which are less than 2011 and have the property that $f(n) = f(2011)$.

*Contare i numeri interi positivi n < 2011 con f(n) = f(2011)*
