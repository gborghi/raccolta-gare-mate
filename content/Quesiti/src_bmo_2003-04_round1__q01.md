*Solve simultaneous equations in four real numbers*

> Solve the simultaneous equations
> $$a + b = c + d,$$
> $$b + c = 2a + b,$$
> $$a + c = b + d,$$
> $$a + b + c = 6$$
> where $a, b, c, d$ are real numbers.
