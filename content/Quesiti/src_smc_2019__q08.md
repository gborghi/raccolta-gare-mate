*Determine which interval contains the solution x of 3^(3^x) = 333*

> The number $x$ is the solution to the equation $3^{(3^x)} = 333$.
> 
> Which of the following is true?
>
> - **(A)** $0 < x < 1$
> - **(B)** $1 < x < 2$
> - **(C)** $2 < x < 3$
> - **(D)** $3 < x < 4$
> - **(E)** $4 < x < 5$
