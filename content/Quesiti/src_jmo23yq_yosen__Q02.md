# Japan Mathematical Olympiad 2013 — Yosen (Preliminary) — Quesito 2

*Two 20x13 grids filled in two orders; count matching cells*

> There are two rectangular grids of $20$ rows and $13$ columns. In each cell of each grid an integer is written as follows:
> 
> - In one grid, the first row is filled from left to right with $1, 2, \ldots, 13$, the second row from left to right with $14, 15, \ldots, 26$, and so on, down to the last row $248, 249, \ldots, 260$.
> - In the other grid, the rightmost column is filled from top to bottom with $1, 2, \ldots, 20$, the second column from the right from top to bottom with $21, 22, \ldots, 40$, and so on, down to the leftmost column $241, 242, \ldots, 260$.
> 
> Find all integers that are written in the same position (same cell) in both grids.
