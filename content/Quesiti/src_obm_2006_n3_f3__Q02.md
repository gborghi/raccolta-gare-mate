# Olimpíada Brasileira de Matemática 2006 — Nível 3 — Fase 3 — Quesito 2

*Bounds an^2 < f(n) < bn^2 for max isosceles triangles*

> Let $n$ be an integer, $n \ge 3$. Define $f(n)$ as the maximum number of isosceles triangles whose vertices belong to some set of $n$ points in the plane with no three points collinear. Prove that there exist positive constants $a$ and $b$ such that $an^2 < f(n) < bn^2$ for every integer $n \ge 3$.
