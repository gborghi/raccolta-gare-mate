*Aquasiperiodic functions on Q, bound on fixed-point-like preimage*

> Let $\mathbb{Q}$ be the set of rational numbers. A function $f : \mathbb{Q} \to \mathbb{Q}$ is called aquasiperiodic if the following property holds: for every $x, y \in \mathbb{Q}$,
> $$f(x + f(y)) = f(x) + y \quad \text{or} \quad f(f(x) + y) = x + f(y).$$
> Show that there exists an integer $c$ such that for any aquasiperiodic function $f$ there are at most $c$ different rational numbers $r$ such that $f(r) = f(-r)$, and find the smallest possible value of $c$.
