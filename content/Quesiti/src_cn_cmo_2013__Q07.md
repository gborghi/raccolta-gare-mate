*Smallest k so that for every valid self-map of a 100-element set there is B with |B|=k and B union f(B)=X*

> Given $X=\{1,2,\ldots,100\}$, consider functions $f:X\to X$ satisfying the following conditions:
> (1) $f(x)\ne x$ for all $x\in X$;
> (2) for any set $A\subseteq X$ with $|A|=40$, we have $A\cap f(A)\ne\varnothing$.
> Find the smallest positive integer $k$ such that for any such function $f$ there exists a set $B\subseteq X$ satisfying $|B|=k$ and $B\cup f(B)=X$.
