*Sum of floor function involving fractional part of square root*

> For each positive integer $n$, define a function
> $$f(n) = \begin{cases} 0, & \text{if } n \text{ is the square of an integer,} \\ \left[\dfrac{1}{\{\sqrt{n}\}}\right], & \text{if } n \text{ is not the square of an integer.} \end{cases}$$
> (Here $[x]$ denotes the maximum integer not exceeding $x$, and $\{x\} = x - [x]$.) Find the value of $\displaystyle\sum_{k=1}^{200} f(k)$.

Trova il valore di $\displaystyle\sum_{k=1}^{200} f(k)$.
