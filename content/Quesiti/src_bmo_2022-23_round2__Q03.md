# British Mathematical Olympiad 2022-23 — Round 2 — Quesito 3

*Count ideal n-lists where running averages are integers*

> For an integer $n \ge 3$, we say that $A = (a_1, a_2, \ldots, a_n)$ is an $n$-list if every $a_k$ is an integer in the range $1 \le a_k \le n$. Let
> $$M_k = \frac{a_1 + a_2 + \cdots + a_k}{k}$$
> for each $k = 1, 2, \ldots, n-1$. We say an $n$-list $A$ is ideal if $M_k$ is an integer for each $k = 1, 2, \ldots, n-1$.
> 
> Find the number of ideal $n$-lists.
