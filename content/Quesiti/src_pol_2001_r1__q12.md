*Sequences with x0=0 and |x_n|=|x_{n-1}+1|; find the minimum of the absolute sum.*

> Consider all sequences $x_0, x_1, \ldots, x_{2000}$ of integers satisfying
> $$x_0 = 0 \quad \text{and} \quad |x_n| = |x_{n-1} + 1| \text{ for } n = 1, 2, \ldots, 2000.$$
> Find the minimum value of the expression $|x_1 + x_2 + \cdots + x_{2000}|$.
