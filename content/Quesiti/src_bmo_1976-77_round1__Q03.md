# British Mathematical Olympiad 1976-77 — Round 1 — Quesito 3

*Inequalities for real numbers and polynomial root exclusion from interval*

> (i) Prove that if $x$, $y$, $z$ are non-zero real numbers $a$, $b$, $c$ then
> $$x(x-y)(x-z) + y(y-x)(y-z) + z(z-x)(z-y) \ge 0.$$
> 
> (ii) Hence or otherwise show that if $u_1, u_2, u_3$ are real numbers and
> $$a^4 + b^4 + c^4 \ge 2(a^2b^2 + b^2c^2 + c^2a^2)$$
> then the equation $a^4 x^4 + b^4 x^3 + c^4 x^2 + b^4 x + a^4 = 0$ has no root in the interval $-1 < x < 3$.
