*2n×2n binary array with 5n zeros; delete n rows and n columns to remove all zeros*

> Given a $2n \times 2n$ array of 0's and 1's containing exactly $5n$ zeros, show that it is possible to remove all the zeros by deleting some $n$ rows and $n$ columns.
