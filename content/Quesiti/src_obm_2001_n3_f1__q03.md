*Count composite numbers in {101, 1001, 10001, ..., 1000000000001}*

> In the set $\{101,\ 1\,001,\ 10\,001,\ \ldots,\ 1\,000\,000\,000\,001\}$ each element is a number formed by the digit 1 at both ends and by digits 0 in between. Some of these elements are prime numbers and others are composite. About the number of composite numbers we can state that:
> 
> **A)** it is equal to 11 $\quad$ **B)** it is equal to 4 $\quad$ **C)** it is less than 3 $\quad$ **D)** it is greater than 4 and less than 11 $\quad$ **E)** it is 3
