# Japan EGMO Selection / Girls MO 2025 — Quesito 1

*Minimum number of distinct values among 14 subset sums*

> Let $a, b, c, d$ be pairwise distinct positive real numbers. Among the following real numbers
> $$a,\quad b,\quad c,\quad d,\quad a+b,\quad a+c,\quad a+d,\quad b+c,\quad b+d,\quad c+d,$$
> $$a+b+c,\quad a+b+d,\quad a+c+d,\quad b+c+d,\quad a+b+c+d$$
> the number of distinct values that appear was exactly $k$. Find the minimum possible value of $k$.
