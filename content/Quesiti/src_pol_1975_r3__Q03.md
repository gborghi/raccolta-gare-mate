*Find smallest positive alpha for which some beta>0 gives sqrt(1+x)+sqrt(1-x)<=2-x^alpha/beta on [0,1], and the smallest such beta*

> Find the smallest positive number $\alpha$ for which there is a positive number $\beta$ such that for all $0 \le x \le 1$,
> $$\sqrt{1+x} + \sqrt{1-x} \le 2 - \frac{x^{\alpha}}{\beta}.$$
> For each such $\alpha$ determine the smallest $\beta > 0$ for which this condition holds.
