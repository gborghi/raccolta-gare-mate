*Find all non-zero reals x,y,z satisfying a system of equations*

> Find all non-zero real numbers $x, y, z$ which satisfy the system of equations:
> $$\left(x^2 + xy + y^2\right)\left(y^2 + yz + z^2\right)\left(z^2 + zx + x^2\right) = xyz,$$
> $$\left(x^4 + x^2y^2 + y^4\right)\left(y^4 + y^2z^2 + z^4\right)\left(z^4 + z^2x^2 + x^4\right) = x^3y^3z^3.$$
