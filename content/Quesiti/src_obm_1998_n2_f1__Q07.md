*Secret code: find a+b+c+d from digit equations*

> In a secret code, the first 10 letters of the alphabet represent the digits 0 through 9, with each letter corresponding to exactly one digit and vice versa. It is known that $d + d = f$, $d \cdot d = f$, $c + c = d$, $c + d = a$, and $a - a = b$. We can conclude that $a + b + c + d$ equals:
> 
> A) 0
> B) 2
> C) 4
> D) 6
> E) 8
