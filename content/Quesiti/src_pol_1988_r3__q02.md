*Expected number of equiprobable permutation indices*

> For a permutation $\pi = (p_1, p_2, \ldots, p_n)$ of $(1, 2, \ldots, n)$ we define $X(\pi)$ to be the number of indices $j$ such that $p_i < p_j$ for all $i < j$. Find the expected value of $X(\pi)$ over all permutations $\pi$ of $(1, 2, \ldots, n)$.
