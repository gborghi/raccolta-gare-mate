*n x n chessboard: count ways to place n^2 numbers in arithmetic progression in each row and column*

> For each integer $n \ge 3$, find the number of ways in which one can place the numbers $1, 2, \ldots, n^2$ in the squares of an $n \times n$ chessboard (one in each square) such that the numbers in each row and in each column form an arithmetic progression.
