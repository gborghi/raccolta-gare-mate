*Prove no subset of R^2 has finite columns and countable rows*

> Prove that there is no set $A\subseteq\mathbb{R}^2$ such that:
> 
> - For every $x\in\mathbb{R}$, the set $\{y\in\mathbb{R}:(x,y)\in A\}$ is finite.
> 
> - For every $y\in\mathbb{R}$, the set $\{x\in\mathbb{R}:(x,y)\in A\}$ is countable (enumerable).
> 
> *Remark:* A set $E$ is called *enumerable* if $E=\emptyset$ or there exists a surjective function $f:\mathbb{N}\to E$.

*Prove che nessun sottoinsieme di R^2 ha colonne finite e righe contabili*

> Provare che non esiste un insieme $A\subseteq\mathbb{R}^2$ che:
> 
> - Per ogni $x\in\mathbb{R}$, l'insieme $\{y\in\mathbb{R}:(x,y)\in A\}$ è finito.
> 
> - Per ogni $y\in\mathbb{R}$, l'insieme $\{x\in\mathbb{R}:(x,y)\in A\}$ è contabile (numeroso).
> 
> < MSK3/>Rimarca: < MSK4/> Un insieme < MSK0/> si chiama < MSK5/>numerable< MSK6/> se < MSK1/> o se esiste una funzione surgettiva < MSK2/>.
