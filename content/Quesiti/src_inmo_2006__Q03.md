# Indian National Mathematical Olympiad 2006 — Quesito 3

*Find all integer triples (a,b,c) with functional equation f(f(a,b,c))=(a,b,c)*

> Let $X$ denote the set of all triples $(a, b, c)$ of integers. Define a function $f : X \to X$ by
> $$f(a, b, c) = (a + b + c,\; ab + bc + ca,\; abc).$$
> Find all triples $(a, b, c)$ in $X$ such that $f(f(a, b, c)) = (a, b, c)$.
