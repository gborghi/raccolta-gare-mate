# Japan Mathematical Olympiad 2006 — Yosen (Preliminary) — Quesito 3

*Count ways to fill 3×4 grid with 1–4 so no value repeats in any row or column*

> In how many ways can each cell of a $3 \times 4$ grid be filled with one of the numbers $1, 2, 3, 4$ so that both of the following conditions are satisfied?
> \begin{itemize}
> \item The same number does not appear more than once in any row.
> \item The same number does not appear more than once in any column.
> \end{itemize}
