*Surjective map on unit disk preserving distance implies isometry*

> Let $D$ be the set of points in $\mathbb{R}^2$ with $|p| \le 1$. Let $f : D \to D$ be a surjective function satisfying $|f(p) - f(q)| \le |p - q|$ for all $p, q \in D$. Prove that $f$ is an isometry, that is, $|f(p) - f(q)| = |p - q|$ for all $p, q \in D$.
> 
> (Notation: $|(x,y)| = \sqrt{x^2 + y^2}$.)
