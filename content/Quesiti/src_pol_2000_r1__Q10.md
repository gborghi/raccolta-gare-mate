*OA, OB, OC pairwise orthogonal unit vectors; line/plane through O with A',B',C' projections of A,B,C; find set of values of OA'^2+OB'^2+OC'^2.*

> Let $\overrightarrow{OA}, \overrightarrow{OB}, \overrightarrow{OC}$ be pairwise orthogonal unit vectors in space. Let $O$ be a variable plane through $O$, and let $A', B', C'$ be the projections of $A, B, C$ onto $O$. Find the set of values of $OA'^2 + OB'^2 + OC'^2$ when $O$ takes all possible positions.
