# Japan Junior Mathematical Olympiad 2023 — Yosen (Preliminary) — Quesito 4

*Max primes among pairwise sums of 5 distinct integers*

> There are $5$ distinct positive integers $a, b, c, d, e$. Among the following $10$ expressions $$a+b,\ a+c,\ a+d,\ a+e,\ b+c,\ b+d,\ b+e,\ c+d,\ c+e,\ d+e,$$ find the maximum possible number whose value is a prime.
