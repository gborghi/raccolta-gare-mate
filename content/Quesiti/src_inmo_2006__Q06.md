*Find integer l with n < l^2 < (1+1/2005)n for n >= 4011^2*

> (a) Prove that if $n$ is a positive integer such that $n \ge 4011^2$, then there exists an integer $l$ such that
> $$n < l^2 < \left(1 + \frac{1}{2005}\right)n.$$
> 
> (b) Find the smallest positive integer $M$ for which whenever an integer $n$ is such that $n \ge M$, there exists an integer $l$ such that
> $$n < l^2 < \left(1 + \frac{1}{2005}\right)n.$$
