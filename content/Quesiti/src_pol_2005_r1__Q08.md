*Lamps on a circle toggled in blocks of k; find k for which all can be turned off.*

> On a circle there are given $n$ lamps, each of which can be either turned on or turned off. A sequence of operations is performed: in every operation one selects $k$ successive lamps and changes the state of each of them. Initially all the lamps are turned on. For a given positive integer $n$, find all positive integers $k$ for which one can turn all the lamps off.
