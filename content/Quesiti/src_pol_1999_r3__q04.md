*Find all n>=2 for which the cyclic system x_i^2+x_{i+1}^2+50=16x_i+12x_{i+1} has an integer solution.*

> Find all integers $n \ge 2$ for which the following system has a solution in integers:
> $$\begin{aligned} x_1^2 + x_2^2 + 50 &= 16x_1 + 12x_2, \\ x_2^2 + x_3^2 + 50 &= 16x_2 + 12x_3, \\ &\cdots \\ x_n^2 + x_1^2 + 50 &= 16x_n + 12x_1. \end{aligned}$$
