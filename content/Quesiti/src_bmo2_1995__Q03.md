*Prove bounds on roots given sum and sum-of-products conditions*

> Let $a$, $b$, $c$ be real numbers satisfying $a < b < c$, $a + b + c = 6$, and $ab + bc + ca = 9$. Prove that $0 < a < 1 < b < 3 < c < 4$.
