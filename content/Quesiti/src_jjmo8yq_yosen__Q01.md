# Japan Junior Mathematical Olympiad 2010 — Yosen (Preliminary) — Quesito 1

*Two-digit number stays 3 digits after multiplying by 7 twice*

> There is a two-digit positive integer. When it is multiplied by $7$, it becomes a three-digit number. When it is multiplied by $7$ again, it is still a three-digit number. How many values are possible for the original integer?
