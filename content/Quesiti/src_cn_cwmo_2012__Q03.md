*Max number of subsets of an n-element set with pairwise intersection or containment*

> Let $E$ be a given set with $n$ elements. Suppose $A_1, A_2, \ldots, A_k$ are $k$ distinct non-empty subsets of $E$, with the property that: for any $1 \le i < j \le k$, either $A_i \cap A_j = \emptyset$ or one includes the other (i.e., $A_i \subseteq A_j$, or $A_j \subseteq A_i$). Find the maximum value of $k$. (posed by Lang Guangqiong)
