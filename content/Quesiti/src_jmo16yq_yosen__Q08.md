# Japan Mathematical Olympiad 2006 — Yosen (Preliminary) — Quesito 8

*Minimum days for Jiro and Saburo to finish reading a 120-page book together*

> There is a book. Jiro reads it at $2$ pages per day, and Saburo reads it at $3$ pages per day. However, when either of them reaches the end of a chapter, he stops for the day even if he has not read the required number of pages. The book has $10$ chapters and $120$ pages in total. Each person starts reading from the first page of a new chapter each day. Find the minimum possible number of days that it can take from when Jiro starts reading until Saburo finishes reading. Note that both may start reading on the same day.
