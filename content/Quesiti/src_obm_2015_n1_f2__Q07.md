*10-digit numbers where each new digit equals sum of all previous (or last digit of that sum); find smallest, largest, possible last digits*

> Janaina likes to write $10$-digit numbers in the following way:
> 
> (i) She writes each digit from left to right, never using a leading zero (the first digit is nonzero);
> 
> (ii) Each new digit she writes must equal the sum of all digits written so far, OR it must equal the last digit of that sum. For example, she can write $5\,386\,248\,624$, because $8 = 5 + 3$, $6$ is the last digit of $5 + 3 + 8 = 16$, $2$ is the last digit of $5 + 3 + 8 + 6 = 22$, and so on. Another example is $4\,600\,000\,000$.
> 
> a) What is the smallest number she can write?
> 
> b) What is the largest number she can write?
> 
> c) Which digits can appear as the last (tenth) digit?
