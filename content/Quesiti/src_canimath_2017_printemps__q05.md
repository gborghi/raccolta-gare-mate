*Maximum consecutive odd numbers when repeatedly adding largest digit*

> A number is written on a board. At each step, the largest digit of the current number is added to it (for example, if $142$ is written, the next number will be $146$). What is the largest number of odd numbers that can appear consecutively when proceeding in this manner?
