# Japan Mathematical Olympiad 2022 — Yosen (Preliminary) — Quesito 6

*Colorings of triangular-tile board with balanced-color condition at each vertex*

> A board consists of $36$ equilateral-triangular tiles arranged in a triangular lattice with $30$ marked interior vertices. Color each tile either red or blue so that at every marked vertex, the number of red tiles sharing that vertex equals the number of blue tiles sharing that vertex. How many such colorings are there?
