*Solve cyclic system of four cubic equations in a,b,c,d,e*

> Solve in real numbers $a, b, c, d, e$ the following system of equations:
> $$3a = (b + c + d)^3, \quad 3b = (c + d + e)^3, \quad 3c = (d + e + a)^3,$$
> $$3d = (e + a + b)^3.$$
