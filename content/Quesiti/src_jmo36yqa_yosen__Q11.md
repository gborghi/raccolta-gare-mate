*Count sequences (a_1,...,a_26) in [1,20] satisfying a partial-sum balance condition for all pairs*

> Find the number of sequences $(a_1, a_2, \ldots, a_{26})$ of integers with $1 \le a_i \le 20$ satisfying the following condition: for any pair of indices $(i, j)$ with $1 \le i < j \le 26$, there exists an index $t$ with $i \le t \le j$ such that
> $$a_1 + a_2 + \cdots + a_t = a_{t+1} + a_{t+2} + \cdots + a_{26}.$$
