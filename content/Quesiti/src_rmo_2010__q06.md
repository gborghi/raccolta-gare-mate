*Count n in {1,...,2010} where floor(n/floor(sqrt(n))) equals floor((n+1)/floor(sqrt(n+1)))*

> For each integer $n \ge 1$, define $a_n = \left[\dfrac{n}{[\sqrt{n}]}\right]$, where $[x]$ denotes the largest integer not exceeding $x$. Find the number of all $n$ in the set $\{1, 2, 3, \ldots, 2010\}$ for which $a_n = a_{n+1}$.
