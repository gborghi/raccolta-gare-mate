*Show s(n) ≤ 2b for n ≤ 2500 and find equality cases*

> Given a positive integer $n$, let $s(n)$ denote the number of positive integers which appear as blocks of consecutive digits in the binary representation of $n$. For example, $s(13) = 6$ because $13 = 1101_2$, which contains as consecutive blocks the binary representations of $13 = 1101_2$, $6 = 110_2$, $5 = 101_2$, $3 = 11_2$, $2 = 10_2$, and $1 = 1_2$.
> 
> Show that if $n \le 2500$, then $s(n) \le 2b$, where $b$ is the number of bits in the binary representation of $n$, and find the values of $n$ for which equality holds.
