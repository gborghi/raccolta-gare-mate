# Japan Mathematical Olympiad 2013 — Yosen (Preliminary) — Quesito 10

*Flip cards via 2013 operations; count face-up cards*

> There are $2013$ cards numbered $0, 1, \ldots, 2012$. Starting from a state in which all cards are face down, the following operation is performed for $i = 1, 2, \ldots, 2013$ in this order. Operation $i$: turn over every card whose number is $\left\lfloor \dfrac{2013 j}{i} \right\rfloor$ for some integer $j$ with $0 \le j \le i-1$ (a face-down card becomes face up, and a face-up card becomes face down). When all operations are finished, how many cards are face up? Here, for a real number $r$, $\lfloor r \rfloor$ denotes the greatest integer not exceeding $r$.
