*Minimum n so n triples cover all consecutive triples in any 1-to-100 sequence*

> Let $n$ be a positive integer. Suppose $n$ triples of integers $(x_1, y_1, z_1),\,(x_2, y_2, z_2),\,\ldots,\,(x_n, y_n, z_n)$, each with $1 \le x_j, y_j, z_j \le 100$, satisfy the following condition:
> 
> \textit{For every infinite sequence $a_1, a_2, \ldots$ of integers with $1 \le a_i \le 100$ for all $i \ge 1$, and for every positive integer $i$, there exists an integer $j$ with $1 \le j \le n$ such that $(a_i, a_{i+1}, a_{i+2}) = (x_j, y_j, z_j)$.}
> 
> Find the minimum possible value of $n$.
