*Define good integer via lattice-point rational/irrational conditions; find minimum and test 2005*

> An integer $n$ is called a good number if $n \ge 3$ and there are $n$ lattice points $P_1, P_2, \cdots, P_n$ in the coordinate plane satisfying the following conditions. If line segment $P_iP_j$ has a rational (irrational) length, then $P_i$ is such that both line segments $P_iP_k$ and $P_iP_l$ have irrational lengths; and if line segment $P_iP_j$ has an irrational length, then there is $P_k$ such that both line segments $P_kP_i$ and $P_kP_j$ have rational lengths. (A point in the coordinate plane is a lattice point if both of its coordinates are integers.)
> (1) Determine the minimum good number.
> (2) Determine if 2005 is a good number.
