*Determine all n for which a floor-pair counting function equals n+1*

> For a positive integer $n$, let $g(n)$ denote the number of pairs of integers $(a, b)$ with $a \ge 1$, $b \ge 1$ and $\left\lfloor \dfrac{n}{a} \right\rfloor = b$.
> 
> Determine all $n$ for which $g(n) = n + 1$.
