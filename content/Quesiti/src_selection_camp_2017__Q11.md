# Japan IMO Selection Camp 2017 — Selection Camp — Quesito 11

*Find all f on positive integers with f(m)+f(n)-mn nonzero and ratio integer*

> Find all functions $f$ defined on the positive integers taking positive integer values such that for any positive integers $m, n$, one has $f(m) + f(n) - mn \neq 0$ and $\dfrac{mf(m) + nf(n)}{f(m) + f(n) - mn}$ is an integer.
