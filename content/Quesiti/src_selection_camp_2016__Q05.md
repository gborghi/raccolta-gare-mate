# Japan IMO Selection Camp 2016 — Selection Camp — Quesito 5

*Find all function pairs satisfying a rational functional equation*

> Find all pairs of functions $(f, g)$ defined on the rationals and taking rational values such that for all rational numbers $x, y$,
> $$f(f(x) + yg(x)) = (x+1)g(y) + f(y).$$
