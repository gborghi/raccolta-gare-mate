*Count 4x4 Latin squares with 2x2 block condition*

> Make a $4 \times 4$ grid of cells and write in the digits from $1$ to $4$, four of each (one digit per cell), subject to the following three conditions:
> (i) In each row, $1, 2, 3, 4$ each appear exactly once.
> (ii) In each column, $1, 2, 3, 4$ each appear exactly once.
> (iii) When the whole grid is divided into four parts by the bold lines as shown in the figure below (the four $2 \times 2$ blocks), each part contains $1, 2, 3, 4$ each exactly once.
> How many such ways of writing in the digits are there?
