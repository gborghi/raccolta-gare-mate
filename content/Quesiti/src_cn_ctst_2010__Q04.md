*Good graph map exists with value at most n factorial*

> Let $G = G(V, E)$ be a simple graph with vertex set $V$ and edge set $E$, and assume that $|V| = n$. A map $f: V \to \mathbb{Z}$ is said to be good if it satisfies:
> (1) $\displaystyle\sum_{v \in V} f(v) = |E|$;
> (2) If one colors arbitrarily some vertices into red, there exists a red vertex $v$ such that $f(v)$ is not greater than the number of vertices adjacent to $v$ that are not colored red.
> Let $m(G) = \min\{\displaystyle\sum_{v} |f(v)| : f \text{ is good}\}$. Show that $m(G) \le n!$.
