# Olimpíada Brasileira de Matemática 2006 — Nível 1 — Fase 2 — Quesito 3

*Find A/223 from a telescoping sum pattern*

> Observe the following equalities:
> $$1 = 1$$
> $$1 + 2 + 1 = 4$$
> $$1 + 2 + 3 + 2 + 1 = 9$$
> $$1 + 2 + 3 + 4 + 3 + 2 + 1 = 16$$
> $$\vdots$$
> $$1 + 2 + 3 + \cdots + 2006 + 2007 + 2006 + \cdots + 3 + 2 + 1 = A$$
> 
> What is the value of $\dfrac{A}{223}$?
