# British Mathematical Olympiad 1989-90 — Round 2 — Quesito 1

*Integer-coefficient polynomial taking value 1990 at four distinct integers cannot take 1991*

> Prove that if
> $$p(x) = a_0 x^n + a_1 x^{n-1} + \cdots + a_n$$
> whose coefficients are integers, takes the value 1990 for four distinct integer values of $x$, then it does not take the value 1991 for any integer value of $x$.
