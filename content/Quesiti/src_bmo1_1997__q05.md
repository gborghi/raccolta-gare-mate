*Three yes/no inequalities for positive x, y with sum conditions*

> Let $x$, $y$ be positive numbers.
> 
> (i) If $x + y \ge 3$, is it necessarily true that $\dfrac{1}{x} + \dfrac{1}{y} \le \dfrac{3}{7}$?
> 
> (ii) If $x + y \ge 3$, is it necessarily true that $\dfrac{1}{x} + \dfrac{1}{y} \le 3$?
> 
> (iii) If $x + y \ge 3$, is it necessarily true that $\dfrac{1}{x} + \dfrac{1}{y} \le 3$?
