# Japan Mathematical Olympiad 2020 — Yosen (Preliminary) — Quesito 3

*Count arrangements of 1-6 in 2x3 grid with adjacent cells differing*

> Fill each cell of a $2 \times 3$ grid with the integers $1, 2, 3, 4, 5, 6$ (each used exactly once) so that any two cells sharing an edge contain different numbers. How many such arrangements are there? Two arrangements that coincide after rotation or reflection are still counted as distinct.
