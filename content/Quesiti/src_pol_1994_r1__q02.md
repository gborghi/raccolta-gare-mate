*Iterated absolute-value functions f_{n+1}=|f_n-2|, f_0=|x|; solve f_n(x)=1*

> The sequence of functions $f_n : \mathbb{R} \to \mathbb{R}$ is given by $f_0(x) = |x|$ and, for each $n$,
> $$f_{n+1}(x) = |f_n(x) - 2| \quad \text{for all } x.$$
> Solve the equation $f_n(x) = 1$, where $n$ is a given positive integer.

Le funzioni di valore assoluto <iterate f_{n+1}=li_f_n-2 khi, f_0=li_x khi; risolvere f_n(x)=1*
