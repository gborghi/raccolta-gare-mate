*Sequence u_n defined by recurrence; find c such that u_i divides u_j whenever i ≤ j.*

> For each positive integer $c$, the sequence $u_n$ of integers is defined by
> $$u_1 = 1, \quad u_2 = c, \quad u_n = (2n+1)u_{n-1} - (n^2-1)u_{n-2}, \quad (n \ge 3).$$
> For which values of $c$ does this sequence have the property that $u_i$ divides $u_j$ whenever $i \le j$?
> 
> (Note: If $x$ and $y$ are integers, then $x$ divides $y$ if and only if there exists an integer $z$ such that $y = xz$. For example, $x = 4$ divides $y = -12$, since we can take $z = -3$.)
