*Sequence satisfying recurrence with fixed parameter k*

> The sequence $u_1, u_2, u_3, \ldots$ satisfies $u_1 = 1$ and
> $$u_{n+1} - u_n = k u_{n-1}, \quad \text{for each } n \ge 2,$$
> where $k$ is some fixed integer. If $u_{2000} = 2000$, determine all possible values of $k$.
