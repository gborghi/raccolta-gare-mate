# Olimpíada Brasileira de Matemática 2008 — Nível 3 — Fase 1 — Quesito 25

*Five positive integers >1 with cyclic product conditions; find a+b+c+d+e*

> Five positive integers $a, b, c, d, e$, all greater than one, satisfy the following conditions:
> $$a(b + c + d + e) = 128$$
> $$b(a + c + d + e) = 155$$
> $$c(a + b + d + e) = 203$$
> $$d(a + b + c + e) = 243$$
> $$e(a + b + c + d) = 275$$
> What is the value of $a + b + c + d + e$?
> 
> (A) $9$ (B) $16$ (C) $25$ (D) $36$ (E) $49$
