# Japan Mathematical Olympiad 2002 — Yosen (Preliminary) — Quesito 12

*Functional equation over rationals; solve f(x)=2002*

> A function $f(x)$ is defined for every rational number and takes rational values, and satisfies the following conditions $(1)$, $(2)$, $(3)$.
> 
> $(1)$ $f(0) = 2,\ f(1) = 3$.
> 
> $(2)$ For any rational number $x$ and any integer $n$, the following holds:
> $$f(x+n) - f(x) = n\{f(x+1) - f(x)\}.$$
> 
> $(3)$ For any nonzero rational number $x$, the following holds:
> $$f(x) = f\left(\frac{1}{x}\right).$$
> 
> Under these conditions, find all rational numbers $x$ satisfying $f(x) = 2002$.
