*Set of n straight lines in the plane where each line meets exactly 10 others; which value cannot be n?*

> There is a set of $n$ straight lines in the plane such that each line intersects exactly ten others.
> 
> Which of the following could NOT be the number of lines in the set?
>
> - **(A)** $11$
> - **(B)** $12$
> - **(C)** $15$
> - **(D)** $16$
> - **(E)** $20$
