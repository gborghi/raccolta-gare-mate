*Compute f(1996) from a telescoping functional equation*

> A function $f$ is defined over the set of positive integers and satisfies
> $$f(1) = 1996$$
> and
> $$f(1) + f(2) + \cdots + f(n) = n^2 f(n) \quad \text{for all } n > 1.$$
> Calculate the value of $f(1996)$.
