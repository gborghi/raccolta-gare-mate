# British Mathematical Olympiad 1980-81 — Round 2 — Quesito 7

*8x>=7y for non-negative integers iff decomposition x=a+2b+3c+7d, y=b+2c+5d exists*

> Prove that if $x$, $y$ are non-negative integers then $8x \geq 7y$ if and only if there exist non-negative integers $a$, $b$, $c$, $d$ such that
> $$x = a + 2b + 3c + 7d, \qquad y = b + 2c + 5d.$$
