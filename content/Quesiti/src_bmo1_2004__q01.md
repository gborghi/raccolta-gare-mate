*Solve simultaneous equations in four real variables*

> Solve the simultaneous equations
> $$ab + c + d = 3, \quad bc + d + a = 5, \quad cd + a + b = 2, \quad da + b + c = 6,$$
> where $a, b, c, d$ are real numbers.
