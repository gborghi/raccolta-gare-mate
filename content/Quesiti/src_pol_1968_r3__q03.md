*n distinct numbers: minimize or maximize number of distinct pairwise sums*

> Given an integer $n \ge 2$, give an example of a set of $n$ mutually different numbers $a_1, \ldots, a_n$ for which the set of their pairwise sums $a_i + a_j$ ($i \ne j$) contains as few different numbers as possible; also give an example of a set of $n$ different numbers $b_1, \ldots, b_n$ for which the set of pairwise sums $b_i + b_j$ ($i \ne j$) contains as many different numbers as possible.
