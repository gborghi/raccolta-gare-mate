*Three quadratics share a common zero; find the polynomials*

> Let $\alpha$ be the common zero of the three polynomials $x^2 + ax + b$, $x^2 + x + ab$ and $ax^2 + x + b$. (Thus $\alpha^2 + a\alpha + b = 0$, $\alpha^2 + \alpha + ab = 0$ and $a\alpha^2 + \alpha + b = 0$.) Find all such polynomials, i.e.\ find all values of $a$ and $b$.
