*Affermazione non deducibile su polinomio a valori interi*

> Il polinomio $ax^2 + bx + c$ assume valori interi per ogni valore intero della variabile $x$. Quale delle seguenti affermazioni non può essere dedotta?
>
> - **(A)** $c$ è intero
> - **(B)** $a + b + c$ è intero
> - **(C)** $a$, $b$, $c$ sono interi
> - **(D)** se $a$ è intero anche $b$ è intero
> - **(E)** $2a$ è intero.

**Area:** [[Algebra e Analisi]]
**Risposta:** C

*Non-deductible claim on polynomial with integer values*

> The $ax^2 + bx + c$ polynomial assumes integer values for each integer value of the variable $x$. Which of the following cannot be deduced?
>
> - **(A)** $c$ is whole
> - **(B)** $a + b + c$ is whole
> - **(C)** < MSK0/>, < MSK1/>, < MSK2/> are integers
> - **(D)** If $a$ is integer $b$ is also integer
> - **(E)** $2a$ is whole.

**Answer:** C
