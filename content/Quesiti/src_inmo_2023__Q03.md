*Find all c >= 1 for which a function f: N -> N satisfies two conditions*

> Let $\mathbb{N}$ denote the set of all positive integers. Find all real numbers $c \geq 1$ for which there exists a function $f : \mathbb{N} \to \mathbb{N}$ satisfying:
> (a) for any $x, n \in \mathbb{N}$, the quantity $\dfrac{f(x+n) - f(x)}{n^c}$ is an integer if and only if $n = 1$;
> (b) for all $x \in \mathbb{N}$, $|f(x+1) - f(x)| < 2023$.
