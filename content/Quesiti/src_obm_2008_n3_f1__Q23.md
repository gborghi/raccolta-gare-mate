# Olimpíada Brasileira de Matemática 2008 — Nível 3 — Fase 1 — Quesito 23

*Multiplicative magic square with cells 4, AC, C, 24; find A+B+C+D*

> Below we have a multiplicative magic square, where the product of the numbers in each row, column and diagonal is the same and equal to the four-digit number $ABCD$, where each letter represents a digit and each cell contains an integer. If $AC$ represents the two-digit number in the center of the square, the sum $A + B + C + D$ is: (In the $3\times 3$ grid: the top-right cell is $4$; the center cell is $AC$; the bottom-middle cell is $C$; and the bottom-right cell is $24$.)
> 
> (A) $17$ (B) $18$ (C) $19$ (D) $20$ (E) $21$
