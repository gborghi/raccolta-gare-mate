*Prove a polynomial with integer coefficients taking value 1990 at four distinct integers cannot take value 1997.*

> Prove that if the polynomial $$p(x) = a_0x^n + a_1x^{n-1} + \ldots + a_{n-1}x + a_n,$$ whose coefficients are integers, takes the value $1990$ for four distinct integer values of $x$, then it cannot take the value $1997$ for any integer value of $x$.
