*Find g(2002) for function satisfying two-sided growth inequalities*

> It is given that $f(x)$ is a function defined on $\mathbf{R}$, satisfying $f(1) = 1$, and for any $x \in \mathbf{R}$,
> $$f(x + 5) \ge f(x) + 5,$$
> $$f(x + 1) \le f(x) + 1.$$
> If $g(x) = f(x) + 1 - x$, then $g(2002) = $ \underline{\quad}.
