*Ordinamento di tre frazioni a,b,c*

> Siano a = 0, 1
> 0, 5, b = 0, 5
> 1 , c =
> 1
> 0, 5; allora
>
> - **(A)** a > b > c
> - **(B)** b > a > c
> - **(C)** c > a > b
> - **(D)** a > c > b
> - **(E)** c > b > a.

*Ordering of three fractions a,b,c*

> If a = 0, 1 0, 5, b = 0, 5 1 , c = 1 0, 5, then
>
> - **(A)** a > b > c
> - **(B)** b > a > c
> - **(C)** c > a > b
> - **(D)** a > c > b
> - **(E)** c > b > a.
