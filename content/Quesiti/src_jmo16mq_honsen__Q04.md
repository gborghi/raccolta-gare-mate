# Japan Mathematical Olympiad 2006 — Honsen (Final) — Quesito 4

*Count lattice path crossing configurations; find all valid tuples*

> Let integers $m, n$ satisfy $2 \le m \le n$. Consider an $m \times n$ grid of streets: there are $m$ horizontal streets running east–west and $n$ vertical streets running north–south. A path is a route that travels only east along horizontal streets. There is a rectangular region of the grid. We say a pair $(a, a')$ with $a \neq a'$ and a pair $(b, b')$ with $b \neq b'$ satisfy the condition if: starting from the west end of horizontal street $a$ (resp.\ $a'$), among all north–south intersections, the crossing points of the path from street $a$ and the path from street $a'$ with north–south streets $b$ and $b'$ (respectively) are such that, counting from the beginning (including the start) to the end, each crossing point is visited exactly once. Find all tuples $(m, n, a, b, a', b')$ of positive integers satisfying this condition.
