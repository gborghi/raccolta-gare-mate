*Sequence defined by recurrence; find which values c make all terms integers*

> For each positive integer $c$, the sequence $u_n$ of integers is defined by
> $$u_1 = 1,\quad u_2 = c,\quad u_n = (2n-3)u_{n-1} - (n-3)u_{n-2} \quad (n \ge 3).$$
> For which values of $c$ is it the case that $u_n$ is an integer for all $n \in \mathbb{Z}^+$?
> 
> (Note: If $x$ and $y$ are integers, then $x$ divides $y$ if and only if there exists an integer $z$ such that $y = xz$. For example, $4$ divides $-12$, since we can take $z = -3$.)
