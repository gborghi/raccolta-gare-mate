*Trigonometric sum, function g, limit condition on a*

> For any positive integer $n$ you are given that
> $$1 + \binom{2n}{1}\cos\theta + \binom{2n}{2}\cos 2\theta + \cdots + \cos 2n\theta = (2\cos\theta)^{2n}\cos n\theta$$
> where there are $2n+1$ terms on the left hand side. All these expressions are defined to be $f(\theta)$. The function $g(\theta)$ is defined by
> $$g(\theta) = 1 + \binom{2n}{1}\cos 2\theta + \binom{2n}{2}\cos 4\theta + \cdots + \cos 2n\theta.$$
> Given that there is no rational $k$ for which $g = k f$, find the values of $a$ for which
> $$\lim_{n \to \infty} \frac{g(a)}{f(a)} = \frac{1}{2}.$$
