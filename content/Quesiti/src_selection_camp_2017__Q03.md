# Japan IMO Selection Camp 2017 — Selection Camp — Quesito 3

*Find all k such that sum of k-th powers equals (sum)^{r_n} for all n*

> Let $a_1, a_2, \ldots$ and $r_1, r_2, \ldots$ be sequences of positive integers. Find all positive integers $k$ such that there exist such sequences satisfying:
> 
> - $a_1 < a_2 < a_3 < \cdots$;
> - For every positive integer $n$,
> $$a_1^k + a_2^k + \cdots + a_n^k = (a_1 + a_2 + \cdots + a_n)^{r_n}.$$
