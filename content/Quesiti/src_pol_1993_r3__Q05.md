*Find all real functions satisfying three functional equations.*

> Find all functions $f : \mathbb{R} \to \mathbb{R}$ satisfying the following conditions:
> $$f(-x) = -f(x) \quad \text{and} \quad f(x+1) = f(x) + 1 \quad \text{for } x \in \mathbb{R};$$
> $$f\!\left(\frac{1}{x}\right) = \frac{f(x)}{x^2} \quad \text{for } x \neq 0.$$
