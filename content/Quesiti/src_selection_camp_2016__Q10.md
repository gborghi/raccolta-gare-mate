# Japan IMO Selection Camp 2016 — Selection Camp — Quesito 10

*Maximum n for snake-like number placement in 2015×2015 grid*

> In a $2015 \times 2015$ grid, the integers $1, 2, \ldots, n$ are written, one in each of $n$ distinct cells, satisfying the following conditions:
> - For every integer $i$ with $1 \le i \le n-1$, the cell containing $i$ and the cell containing $i+1$ share an edge, and the cell containing $n$ and the cell containing $1$ share an edge. Moreover, no two cells that both contain numbers share an edge except for those pairs described above.
> - For any integers $i, j$ with $1 \le i < j \le n$, if the cell containing $i$ and the cell containing $j$ share only a corner (vertex), then $\min\{j - i,\, n + i - j\} = 2$.
> 
> Find the maximum possible value of $n$.
