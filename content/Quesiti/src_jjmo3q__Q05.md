# Japan Junior Mathematical Olympiad 2005 — Quesito 5

*Same digit-count in base ten and base five*

> In base ten, $110$ is written as $420$ in base five, which is also a $3$-digit number. In this way, how many positive integers in total have the same number of digits when written in base ten as when written in base five?
