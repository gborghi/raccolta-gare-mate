*Three degree-one polynomials whose n-th powers satisfy u1^n+u2^n=u3^n are all proportional to Ax+B.*

> Polynomials $u_i(x) = a_i x + b_i$ ($a_i, b_i \in \mathbb{R}$, $i = 1, 2, 3$) satisfy
> $$u_1(x)^n + u_2(x)^n = u_3(x)^n \quad \text{for some integer } n \ge 2.$$
> Prove that there exist real numbers $A$, $B$, $c_1$, $c_2$, $c_3$ such that $u_i(x) = c_i(Ax + B)$ for $i = 1, 2, 3$.
