*Cover m integers using a small set T plus shifts in [-n,n]*

> Given two integers $m$, $n$ greater than 1 and integers $a_1 < a_2 < \cdots < a_m$, prove that there exists a set $T$ of integers with $|T| \le 1 + \dfrac{8n}{m-1}$ such that each $a_i$ can be written as $a_i = t + s$ for some $t \in T$ and some $s \in \{-n, -n+1, \ldots, n\}$. (Posed by Leng Gangsong)
