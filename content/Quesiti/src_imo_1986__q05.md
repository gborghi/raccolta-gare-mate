*Find all functions with f(xf(y))f(y)=f(x+y) and f(2)=0*

> Find all functions f, defined on the non-negative real numbers and taking nonnegative real values, such that:
> (i)
> f(xf(y))f(y) = f(x + y) for all x, y ≥0,
> (ii)
> f(2) = 0,
> (iii)
> f(x) ̸= 0 for 0 ≤x < 2.
