*Type-T_n trinomial functions on [-1,1]: show a piecewise function is of some type T_N*

> We call functions of type $T_0$ the functions « trinomials » on $[-1,1]$, defined by:
> $$t : [-1,1] \to \mathbb{R}, \quad x \mapsto t(x) = ax^2 + bx + c$$
> where $a$, $b$, $c$ are arbitrary real numbers. For every natural number $n$, we call functions of type $T_n$ the functions of the form $f + \lambda g$, where $\lambda$ is an arbitrary real number and $f$, $g$ are functions of type $T_{n-1}$.
> 
> **1.** Establish that the function $\varphi$, defined by $\varphi(x) = 0$ for $x \in [-1, 0[$ and $\varphi(x) = x$ for $x \in [0, 1]$, is of type $T_1$.
> 
> **2.** Consider two trinomial functions $t_1$ and $t_2$ such that $t_1(0) = t_2(0)$, and define the function $f : [-1, 1] \to \mathbb{R}$ by:
> For every real $x$ in $[-1, 0[$, $f(x) = t_1(x)$, and for every real $x$ in $[0, 1]$, $f(x) = t_2(x)$.
> 
> Show that there exists a natural number $N$ such that the function $f$ is of type $T_N$.
