*Divide sets into 3-element subsets with equal element sums*

> (1) Can one divide the set $\{1, 2, \ldots, 96\}$ into 32 subsets, each containing three elements, and the sums of the three elements in each subset are all equal?
> 
> (2) Can one divide the set $\{1, 2, \ldots, 99\}$ into 33 subsets, each containing three elements, and the sums of the three elements in each subset are all equal? (Posed by Liu Shixiong)
