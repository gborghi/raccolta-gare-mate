# British Mathematical Olympiad 1998-99 — Round 1 — Quesito 5

*Find all p such that f(1999)=p for a bijective function with a recurrence condition*

> Consider all functions $f$ from the positive integers to the positive integers such that
> (i) for each positive integer $m$, there is a unique positive integer $n$ such that $f(n) = m$;
> (ii) for each positive integer $n$, we have
> $$f(n+1) \text{ is either } 4f(n) - 1 \text{ or } f(n) - 1.$$
> Find the set of positive integers $p$ such that $f(1999) = p$ for some function $f$ with properties (i) and (ii).
