*Find all functions where a given inequality holds for all integers*

> Find all functions $f$ defined on the positive integers taking positive integer values such that for any integers $m, n \ge 1$,
> $$m^2 + f(n)^2 + (m - f(n))^3 \ge f(m)^2 + n^2$$
> holds.
