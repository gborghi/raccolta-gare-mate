# Japan IMO Selection Camp 2025 — Selection Camp — Quesito 1

*Find c so every finite set has a,b,e with e dividing a+b+2c*

> Find all positive integers $c$ with the following property: for every nonempty finite set $\mathcal{S}$ of positive integers, there exist (not necessarily distinct) elements $a, b$ of $\mathcal{S}$ and an element $e$ of $\mathcal{S}$ such that $a + b + 2c$ is divisible by $e$.
