# Coupe Animath Automne 2014 — Quesito 7

*Digit-rule sequence; can a_0 be chosen so the sequence is infinite?*

> Let $a_0$ be a strictly positive integer. From it we construct a sequence of integers $a_1, a_2, \cdots$ according to the following procedure:
> 
> - if the units digit of $a_i$ is strictly greater than $5$, then $a_{i+1} = 9 a_i$;
> - if the units digit of $a_i$ is less than or equal to $5$ and nonzero, then the number thus formed (by erasing that units digit) is $a_{i+1}$.
> 
> Clearly, if one has erased all the digits of $a_i$, the procedure stops and there is no $a_{i+1}$.
> 
> Can one choose $a_0$ so that the sequence thus constructed is infinite?
