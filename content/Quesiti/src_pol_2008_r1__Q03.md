*For the integer sequence a_1=1, a_2=2, a_n=3a_{n-1}+5a_{n-2}, decide if some k>=2 divides every a_n.*

> A sequence $(a_n)$ of integers is defined by $a_1 = 1$, $a_2 = 2$ and
> $$a_n = 3a_{n-1} + 5a_{n-2} \quad \text{for } n = 3, 4, 5, \ldots .$$
> Does there exist an integer $k \ge 2$ for which $k$ divides $a_n$ for all $n = 1, 2, \ldots$?
