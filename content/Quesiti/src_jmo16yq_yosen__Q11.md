# Japan Mathematical Olympiad 2006 — Yosen (Preliminary) — Quesito 11

*Find minimum constant c for functional inequality involving f(x,y) on unit square*

> For real numbers $x, y$ with $0 \le x, y \le 1$, define
> $$f(x, y) = xy\sqrt{1 - x^2} - x^2 y\sqrt{1 - y^2}.$$
> Find the minimum constant $c$ such that: for every integer $n \ge 2$ and every real numbers $0 \le a_1 < a_2 < \cdots < a_n \le 1$,
> $$f(a_1, a_2) + f(a_2, a_3) + \cdots + f(a_{n-1}, a_n) < c.$$
