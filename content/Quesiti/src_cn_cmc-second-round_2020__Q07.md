*Two positive n-term sequences with equal sums and pairwise product-dominates-sum; find minimum total sum*

> (50 marks) Given integer $n \ge 2$, let $a_1, a_2, \ldots, a_n, b_1, b_2, \ldots, b_n > 0$, satisfying
> $$a_1 + a_2 + \cdots + a_n = b_1 + b_2 + \cdots + b_n$$
> and there is always $a_i a_j \ge b_i + b_j$ for any $i, j$ $(1 \le i < j \le n)$. Find the minimum value of $a_1 + a_2 + \cdots + a_n$.
