*Count permutations of {1,...,6n-1} satisfying two index-difference monotonicity conditions.*

> For each natural number $n$ find the number of permutations $(x_1,x_2,\ldots,x_{6n-1})$ of the set $\{1,2,\ldots,6n-1\}$ with the following properties:
> 
> (i) If $i-j=2n+1$, then $x_i>x_j$;
> 
> (ii) If $i-j=4n$, then $x_i<x_j$.
