*Determine all n with q(n) > q(n+1) where q(n)=floor(n/floor(sqrt n))*

> For any real number $x$, let $\lfloor x \rfloor$ denote the greatest integer which is less than or equal to $x$. Define the function $q$ on the positive integers by
> $$q(n) = \left\lfloor \frac{n}{\lfloor \sqrt{n} \rfloor} \right\rfloor \quad \text{for } n = 1, 2, 3, \ldots$$
> Determine all positive integers $n$ for which $q(n) > q(n+1)$.

> Per qualsiasi numero reale $x$, $\lfloor x \rfloor$ indica il numero intero più grande inferiore o uguale a $x$.
