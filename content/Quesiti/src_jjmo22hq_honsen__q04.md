*Maximum number of white cells in 2024×2024 colored grid*

> A $2024 \times 2024$ grid is given. Each cell is colored red, blue, or white, with every cell receiving exactly one color. The coloring satisfies the following condition: each red cell has exactly $1$ red cell among its (at most $4$) neighbors sharing a side; and for every $2 \times 2$ square of cells, the number of blue cells it contains is either $1$ or $2$. Find the maximum possible number of white cells.
