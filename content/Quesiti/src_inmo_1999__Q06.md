# Indian National Mathematical Olympiad 1999 — Quesito 6

*Split set {1,2,...,4n} into disjoint four-element subsets with equal element sum*

> For which positive integers $n$ can the set $\{1, 2, \ldots, 4n\}$ be split into $n$ disjoint four-element subsets $\{a, b, c, d\}$ such that in each of them $a = \frac{b+c+d}{3}$?
