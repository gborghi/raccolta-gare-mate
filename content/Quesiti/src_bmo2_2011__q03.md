*Count positive integers below 2011 sharing f-value with 2011*

> The function $f$ is defined on the positive integers as follows:
> $$f(1) = 1,$$
> $$f(2n) = 2f(n) \text{ if } n \text{ is even},$$
> $$f(2n) = 2f(n) \text{ if } n \text{ is odd},$$
> $$f(2n+1) = 2f(n) \text{ if } n \text{ is even},$$
> $$f(2n+1) = 2f(n)+1 \text{ if } n \text{ is odd}.$$
> Find the number of positive integers $v$ which are less than $2011$ and have the property that $f(v) = f(2011)$.

*Contare i numeri interi positivi al di sotto del 2011 condividendo il valore f con il 2011*

> La funzione $f$ è definita sui numeri interi positivi come segue: $$f(1) = 1,$$ $$f(2n) = 2f(n) \text{ if } n \text{ is even},$$ $$f(2n) = 2f(n) \text{ if } n \text{ is odd},$$ $$f(2n+1) = 2f(n) \text{ if } n \text{ is even},$$ $$f(2n+1) = 2f(n)+1 \text{ if } n \text{ is odd}.$$ Trova il numero di numeri interi positivi $v$ che sono inferiori a $2011$ e hanno la proprietà di $f(v) = f(2011)$.
