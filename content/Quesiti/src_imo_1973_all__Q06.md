# IMO 1973 (Fifteenth International Olympiad) — Quesito 6

*Construct b_k bounding a_k with ratio and sum conditions*

> Siano $a_1, a_2, \ldots, a_n$ $n$ numeri positivi e sia $q$ un numero reale dato con $0 < q < 1$. Trovare $n$ numeri $b_1, b_2, \ldots, b_n$ tali che:
> 
> (a) $a_k < b_k$ per $k = 1, 2, \ldots, n$,
> 
> (b) $q < \dfrac{b_{k+1}}{b_k} < \dfrac{1}{q}$ per $k = 1, 2, \ldots, n-1$,
> 
> (c) $b_1 + b_2 + \cdots + b_n < \dfrac{1+q}{1-q}(a_1 + a_2 + \cdots + a_n)$.
