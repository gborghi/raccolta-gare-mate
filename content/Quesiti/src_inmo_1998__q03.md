*Divisibility by 5 for a polynomial evaluated at an integer*

> Let $p, q, r, s$ be four integers with $5 \mid s$. If there is an integer $a$ for which $pa^2 + qa + r$ is divisible by 5, prove that there is an integer $b$ such that $sb^2 + rb + q$ is also divisible by 5.
