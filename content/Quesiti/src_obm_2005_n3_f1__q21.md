*Order three large/exotic powers A, B, C by size*

> Let $A = 10^{(16/2005)^2}$, $B = 2005^2$ and $C = 2^{\sqrt{2005}}$. Then:
> (A) $A < B < C$ (B) $A < C < B$ (C) $B < A < C$ (D) $B < C < A$ (E) $C < A < B$
