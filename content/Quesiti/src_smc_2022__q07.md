*Find the minimum total number of stamps when buying first-class (85p) and second-class (66p) stamps, one of each type at a time, so the total cost is an exact integer number of pounds.*

> In 2021, a first class postage stamp cost 85p and a second class postage stamp cost 66p. In order to buy an exact number of each type, what is the smallest total number of stamps that should be purchased?
>
> - **(A)** 10
> - **(B)** 8
> - **(C)** 7
> - **(D)** 5
> - **(E)** 2
