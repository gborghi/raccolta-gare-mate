*Irrational alpha such that beta in A(alpha) implies beta/alpha is an integer*

> For a real number $r$, let $\lfloor r \rfloor$ denote the greatest integer not exceeding $r$. For a positive real number $x$, define the set
> $$A(x) = \{\lfloor nx \rfloor \mid n \text{ is a positive integer}\}.$$
> Find all irrational numbers $\alpha > 1$ such that: whenever $\beta$ is a positive real with $A(\alpha) \supset A(\beta)$, the ratio $\dfrac{\beta}{\alpha}$ is an integer.
