*Positive reals a,b,c,x,y,z; a+b+c=x+y+z=xyz; prove a=x,b=y,c=z*

> Let $a, b, c, x, y, z$ be positive real numbers such that $a + b + c = x + y + z = xyz$. Further, suppose that $a \le x < y \le b < c \le z$ and $a < b < c$. Prove that $a = x$, $b = y$ and $c = z$.
