# Japan Mathematical Olympiad 2012 — Honsen (Final) — Quesito 2

*Find all real functions satisfying a functional equation*

> Find all functions $f$ defined on $\mathbb{R}$ and taking real values such that
> $$f(f(x+y)f(x-y)) = x^2 - yf(y)$$
> holds for all real numbers $x, y$.
