*Polinomio con p(x+1)-p(x-1)=p(x)*

> Il polinomio p è definito da p(x) = ax2005 + x + b, con a e b numeri reali. Per quali
> valori di a e b si ha che p(x + 1) −p(x −1) = p(x) per ogni valore reale di x?
>
> - **(A)** a = 0, b = 2,
> - **(B)** a qualunque e b = 0,
> - **(C)** a = 1 e b qualunque,
> - **(D)** a = b = 0,

*Polynomial with p(x+1)-p(x-1)=p(x) *

> The polynomial p is defined by p(x) = ax2005 + x + b, with a and b real numbers. For what values of a and b do we have that p(x + 1) −p(x −1) = p(x) for every real value of x?
>
> - **(A)** a = 0, b = 2,
> - **(B)** to any and b = 0,
> - **(C)** a = 1 and b any,
> - **(D)** a = b = 0,
> - **(E)** for no values of a and b.

**Answer:** E
