*Largest number obtainable from calculator tree diagram with keys A and B*

> A certain machine has a display showing a positive integer $x$ and two keys A and B. When key A is pressed, the number on the display is replaced by $2x + 1$. When key B is pressed, the number on the display is replaced by $x - 1$. If the display initially shows 5 and we press key A (the number becomes $2 \times 5 + 1 = 11$), then pressing key B replaces it by $x - 1$. The tree diagram of results that can be obtained is shown in the figure. The largest number of two digits that can be obtained is:
> 
> (A) 85 \quad (B) 87 \quad (C) 92 \quad (D) 95 \quad (E) 96
