*Compare counts of good vs second-good operation groups*

> Let $a_1, a_2, \cdots, a_{11}$ be 11 distinct positive integers and write the numbers $1, 2, \cdots, 2007$ in order on the blackboard. Now we define a group of 22 ordered operations.
> 
> The $i$th operation is to take any number on the blackboard, and add $a_i$ to it if $1 \leq i \leq 11$, or add $-a_{i-11}$ to it if $12 \leq i \leq 22$, then write the results on the blackboard. If the final result is an even permutation of $1, 2, \cdots, 2007$, then we call it a 'good' group. If the final result is an odd permutation of $1, 2, \cdots, 2007$, then we call it a 'second good' group.
> 
> Our question is: Which is greater? The number of 'good' groups or that of 'second good' groups? And by how many more?
> 
> (Remark: Suppose $x_1, x_2, \cdots, x_{2007}$ is a permutation of $1, 2, \cdots, 2007$. We call it an even (odd) permutation if $\prod_{1 \leq i < j \leq 2007}(x_i - x_j) > 0$ ($< 0$), and otherwise an odd permutation.)
