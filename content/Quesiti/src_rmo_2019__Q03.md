# Regional Mathematical Olympiad 2019 — Quesito 3

*Prove an inequality for positive reals with a+b+c=1.*

> Let $a, b, c$ be positive real numbers such that $a + b + c = 1$. Prove that
> $$\frac{a}{a^2 + b^3 + c^3} + \frac{b}{b^2 + c^3 + a^3} + \frac{c}{c^2 + a^3 + b^3} \le \frac{1}{5abc}.$$
