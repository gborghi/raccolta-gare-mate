# Olimpíada Brasileira de Matemática 2012 — Nível 2 — Fase 2 — Quesito 8

*Sum of hexadecimal numbers (1)_16 through (100)_16 evaluated in base 10*

> On the planet *Hexaterra*, the most used base is hexadecimal (base 16) instead of decimal (base 10). In base 16, the letters $A, B, C, D, E, F$ are used as digits (in increasing order). Thus, for example, $(10)_{16} = 10 \times 16 + 1 = 176$, $(AB)_{16} = 10 \times 16 + 11 = 176$, $(F0E)_{16} = 15 \times 16^2 + 0 \times 16 + 14 = 3854$. Determine the value of the sum
> $$(1)_{16} + (2)_{16} + \cdots + (D)_{16} + (E)_{16} + (F)_{16} + (10)_{16} + \cdots + (100)_{16}$$
> in base 10.
