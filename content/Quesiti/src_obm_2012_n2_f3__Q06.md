*Find m,n,d values allowing full marking of m×n chocolate bar*

> Maria has a chocolate bar of size $m \times n$ divided into $1 \times 1$ squares. She wishes to mark each small square using the following stamping tool:
> 
> The tool can be used horizontally or vertically. It marks two squares leaving between them two squares with distance $d = 1$; the marked squares cannot be altered and it is not permitted to mark a square more than once. For which values of $m$, $n$, and $d$ is it possible to mark all squares following these conditions?
> 
> *Note:* Example of marking with $d = 3$, using it once vertically and once horizontally.
