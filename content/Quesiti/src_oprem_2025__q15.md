*Generating function f(x)=3x+sqrt(8x^2+1); verify identity; f(n) balanced if n is; sequence u_1=6 strictly increasing*

> **Exercise 3 - For candidates of the general stream NOT following the 'spe maths' specialty, and ALL technological-stream candidates**
> 
> **5. A generating function.** Consider the function $f$ defined on $\mathbb{R}$ by
> $$f(x) = 3x + \sqrt{8x^2+1}.$$
> 
> **(a)** Verify that for every real $x$, $8(f(x))^2 + 1 = (8x + 3\sqrt{8x^2+1})^2$.
> 
> **(b)** Deduce that if $n$ is a balanced number, then $f(n)$ is also.
> 
> Set $u_1 = 6$ and, for every integer $k \ge 1$, $u_{k+1} = f(u_k)$.
> 
> **(c)** Show that $(u_k)_{k \ge 1}$ is a strictly increasing sequence of balanced numbers.

*Funzione di generazione f(x) = 3x+sqrt(8x^2+1); verifica l'identità; f(n) bilanciato se n è; sequenza u_1=6 stringentemente in aumento*

> **Esercizio 3 - Per i candidati del flusso generale NON che seguono la specialità "spe mathematics", e TUTTI i candidati del flusso tecnologico**
> 
> **5. Una funzione generatrice.** Considera la funzione $f$ definita su $\mathbb{R}$ da $$f(x) = 3x + \sqrt{8x^2+1}.$$
> 
> **(a) ** Verificare che per ogni reale $x$, $8(f(x))^2 + 1 = (8x + 3\sqrt{8x^2+1})^2$.
> 
> **(b) ** Deduci che se $n$ è un numero equilibrato, allora $f(n)$ è anche.
> 
> Set $u_1 = 6$ e, per ogni numero intero $k \ge 1$, $u_{k+1} = f(u_k)$.
> 
> **(c) ** Mostra che $(u_k)_{k \ge 1}$ è una sequenza di numeri bilanciati in stretta crescita.
