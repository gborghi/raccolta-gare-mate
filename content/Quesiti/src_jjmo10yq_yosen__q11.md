*Count box dimensions a<=b<=c for a×b×c box built from unit cubes with equal unpainted count*

> Unit cubes are assembled into an $a \times b \times c$ ($a, b, c$ positive integers) rectangular box. Each unit cube on the surface of the box is painted. The number of unit cubes that are not painted equals the number of unit cubes on the surface that are painted. Find the number of triples $(a, b, c)$ with $a \le b \le c$ satisfying this condition. (Triples that are merely rearrangements of each other are counted as one.)
