# Olimpíada Brasileira de Matemática 2025 — Nível 2 — Fase x — Quesito 6

*Minimum size of an obstructive diagonal set in triangulations of convex n-gon*

> Let $\mathcal{P}$ be a convex polygon with $n$ vertices, $n \ge 4$. We can partition $\mathcal{P}$ into triangles by drawing diagonals that do not intersect in the interior of $\mathcal{P}$. A partition of this type is called a \emph{triangulation}. A set $X$ of diagonals of $\mathcal{P}$ is called \emph{obstructive} if it has the following property: any triangulation of $\mathcal{P}$ uses at least one diagonal belonging to $X$.
> 
> Determine, as a function of $n$, the minimum possible number of elements of an obstructive set.
