*Maximum size of completely divisible set containing 2016, elements below 2 million*

> A set is called \emph{completely divisible} if for any two elements $a < b$ in the set, $a$ divides $b$. A set of positive integers $A$ is completely divisible, contains $2016$ as one of its elements, and all elements of $A$ are less than $2{,}000{,}000$. What is the maximum number of elements that $A$ can have?
