*Prove invertibility of binomial coefficient matrix for two index sequences*

> Prove that for any non-negative integers $0 \le i_1 < i_2 < \cdots < i_k$ and $0 \le j_1 < j_2 < \cdots < j_k$, the $k \times k$ matrix $A = (a_{rs})_{1 \le r,s \le k}$ defined by $$a_{rs} = \binom{i_r + j_s}{i_r} = \frac{(i_r + j_s)!}{i_r!\, j_s!}, \quad 1 \le r, s \le k,$$ is invertible.

*Prove l'inversibilità della matrice di coefficienti binomiali per due sequenze di indici*

> Prove che per tutti gli integri non negativi $0 \le i_1 < i_2 < \cdots < i_k$ e $0 \le j_1 < j_2 < \cdots < j_k$, la matrice $k \times k$ $A = (a_{rs})_{1 \le r,s \le k}$ definita da $$a_{rs} = \binom{i_r + j_s}{i_r} = \frac{(i_r + j_s)!}{i_r!\, j_s!}, \quad 1 \le r, s \le k,$$ è invertibile.
