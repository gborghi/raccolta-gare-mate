# IMO 1974 (Sixteenth International Olympiad) — Quesito 4

*Max rectangles partitioning chessboard with balanced colors*

> Si considerino le decomposizioni di una scacchiera $8 \times 8$ in $p$ rettangoli non sovrapposti soggetti alle seguenti condizioni:
> 
> (i) Ogni rettangolo ha tanti quadrati bianchi quanti quadrati neri.
> 
> (ii) Se $a_i$ è il numero di quadrati bianchi nel $i$-esimo rettangolo, allora $a_1 < a_2 < \cdots < a_p$.
> 
> Trovare il valore massimo di $p$ per cui tale decomposizione è possibile. Per questo valore di $p$, determinare tutte le possibili successioni $a_1, a_2, \ldots, a_p$.
