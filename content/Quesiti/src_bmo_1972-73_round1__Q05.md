# British Mathematical Olympiad 1972-73 — Round 1 — Quesito 5

*Infinite set of integers of form 2^n - 3 with no member divisible by the others' primes*

> There is an infinite set of positive integers of the form $2^n - 3$ with only property $Q$: no member is divisible by a prime factor of any other member.
> 
> An outline of a proof is as follows.
> 
> Suppose there is a finite set $S = \{2^{n_1} - 3,\, 2^{n_2} - 3,\, 2^{n_3} - 3, \ldots\}$ with only property $Q$ and $0$ members. Let the prime factors of these numbers be $p_1, p_2, \ldots, p_k$. Consider the number $N = (p_1 - 1)(p_2 - 1)\cdots(p_k - 1) + 1$. By Fermat's theorem $2^{p-1} \equiv 1 \pmod{p}$ for every prime $p$ that does not divide $N$. Hence $2^N \equiv -3 \pmod{p_i}$, and $2^N - 3$ may be added to $S$ to give a larger set with property $Q$.
> 
> Give a properly expanded and reasoned proof that there is an infinite set of positive integers of the form $2^n - 3$ with property $Q$.
