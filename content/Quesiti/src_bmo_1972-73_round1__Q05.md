# British Mathematical Olympiad 1972-73 — Round 1 — Quesito 5

*Infinite set of integers 2^k - 3 with no common prime factor*

> There is an infinite set of positive integers of the form $2^k - 3$ with the property $Q$: no two members of the set have a common prime factor. An outline of a proof is as follows.
> 
> Suppose there is a finite set $S = \{2^{r_1} - 3,\ 2^{r_2} - 3,\ \ldots\}$ with property $Q$ and having $k$ members. Let the prime factors of these $k$ numbers be $p_1, p_2, \ldots, p_j$. Consider the number $N = 2^{(p_1-1)(p_2-1)\cdots(p_j-1)} + 1$. By Fermat's theorem $2^{p-1} \equiv 1 \pmod{p}$ for every prime $p$ that does not divide $a$. Hence $N \equiv -3 + 3 \equiv 1 \pmod{p_s}$, $s = 1$ to $r$, and $N - 3$ may be added to $S$ to give a larger set with property $Q$.
> 
> Give a properly expanded and reasoned proof that there is an infinite set of positive integers of the form $2^k - 3$ with property $Q$.
