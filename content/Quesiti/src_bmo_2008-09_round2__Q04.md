*Show k(n) le 2009 implies n le 2^20, find equality cases*

> Given a positive integer $n$, let $k(n)$ denote the number of positive integers whose binary representations occur as blocks of consecutive digits in the binary representation of $n$.
> 
> For example, if $n = 10$ then the binary representation is $1010$, so $k(10) = 10$ (counting all distinct positive integers representable by a contiguous block of digits of $1010$, noting that $0$ does not count as a positive integer).
> 
> Show that if $k(n) \le 2009$, then $n \le 2^{20}$, and determine the values of $n$ for which equality holds.
