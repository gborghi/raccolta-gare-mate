*Costruire triangolo dati b, c e angolo AMB con condizione*

> Construct triangle $ABC$ if $AC = b$, $AB = c$ and $\angle AMB = \omega$, where $M$ is the midpoint of segment $BC$ and $\omega < 90^\circ$. Prove that a solution exists if and only if
> $$b \tan\frac{\omega}{2} \leq c < b.$$
> In what case does equality hold?
