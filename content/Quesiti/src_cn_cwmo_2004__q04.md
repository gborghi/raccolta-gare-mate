*Find all non-negative c with d(n)+p(n)=n+c for some positive integer n*

> Let $n \in \mathbb{N}$ (the set of positive integers), and $d(n)$ be the number of positive divisors of $n$. Next, $p(n)$ denotes the number of integers in the closed interval $[1, n]$ which are co-prime with $n$. Find all non-negative integers $c$ such that there exists $n \in \mathbb{N}$ satisfying $d(n) + p(n) = n + c$.
