# Pre-Regional Mathematical Olympiad (PRMO) 2015 — Set B — Quesito 17

*Expression P with a+b+c=0; find P*

> Let $a$, $b$, $c$ be such that $a + b + c = 0$ and
> $$P = \frac{a^2}{2a^2 + bc} + \frac{b^2}{2b^2 + ca} + \frac{c^2}{2c^2 + ab}$$
> is defined. What is the value of $P$?
