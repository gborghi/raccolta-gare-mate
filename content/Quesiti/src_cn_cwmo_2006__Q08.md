*Min size of X so a subset Y of size n hits each 2-element set at most once*

> Given a positive integer $n \geq 2$, let $B_1, B_2, \cdots, B_n$ denote $n$ arbitrary subsets of set $X$, each of which contains exactly two elements. Find the minimum value of $|X|$ such that there exists a subset $Y$ of $X$ satisfying:
> (a) $|Y| = n$;
> (b) $|Y \cap B_i| \leq 1$ for $i = 1, 2, \cdots, n$.
