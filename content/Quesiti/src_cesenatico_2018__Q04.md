# Olimpiade Italiana (Cesenatico) 2018 - testi — Quesito 4

*doppio di primo o potenza di primo*

> Sia N un intero maggiore di 1. Chiamiamo x il più piccolo intero positivo con la seguente proprietà: esiste
> un intero positivo y strettamente minore di x −1 tale che x divide N + y. Dimostrare che x è il doppio di un
> numero primo o una potenza di un numero primo.
> Nota: si ricorda che x è una potenza di un numero primo se esistono un primo p ed un intero positivo n ≥1 tali che
> x = pn.
