# IMO 1985 (26th International Mathematical Olympiad) — Quesito 6

*Unique starting value keeping recurrence sequence in (0,1)*

> Per ogni numero reale $x_1$, si costruisce la successione $x_1, x_2, \ldots$ ponendo
> 
> $$x_{n+1} = x_n \left( x_n + \frac{1}{n} \right)$$
> 
> per ogni $n \geq 1$. Dimostrare che esiste esattamente un valore di $x_1$ per cui
> 
> $$0 < x_n < x_{n+1} < 1$$
> 
> per ogni $n$.
