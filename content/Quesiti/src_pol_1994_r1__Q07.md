*Similar triangles built outside quadrilateral ABCD; if ABCD parallelogram then PQRS is*

> Outside a convex quadrilateral $ABCD$, similar triangles $APB, BQC, CRD, DSA$ are constructed so that
> $$\angle PAB = \angle QBC = \angle RCD = \angle SDA, \quad \angle PBA = \angle QCB = \angle RDC = \angle SAD.$$
> Prove that if $ABCD$ is a parallelogram, then so is $PQRS$.
