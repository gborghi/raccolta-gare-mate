# Indian National Mathematical Olympiad 2023 — Quesito 2

*101 cyclic polynomials in positive reals; show not all can have all real roots*

> Suppose $a_0, a_1, \ldots, a_{100}$ are positive reals. Consider the following polynomial for each $k$ in $\{0, 1, \ldots, 100\}$:
> $$a_{101+k} x^{100} + 100a_{k+1} x^{99} + a_{k+2} x^{98} + \cdots + a_{k+99} x + a_k,$$
> where indices are taken modulo 101, i.e., $a_{101m} = a_0$ for any $i$ in $\{1, 2, \ldots, 100\}$.
> 
> Show that it is impossible that each of these 101 polynomials has all its roots real.
