*Function f assigns to each real x the smaller element of {x+1, (15-x)/2}. Find the maximum value of f(x).*

> The function $f$ associates to each real number $x$ the smallest element of the set $\left\{x+1,\ \dfrac{15-x}{2}\right\}$. The maximum value of $f(x)$ is:
> 
> A) $4$
> 
> B) $5$
> 
> C) $\dfrac{11}{2}$
> 
> D) $\dfrac{16}{3}$
> 
> E) $\dfrac{19}{4}$
