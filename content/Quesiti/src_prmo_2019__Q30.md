*Number of elements in E partitioned into 3 subsets with equal sums*

> Let $E$ denote the set of all natural numbers $n$ such that $3 < n < 100$ and the set $\{1, 2, 3, \ldots, n\}$ can be partitioned into 3 subsets with equal sums. Find the number of elements of $E$.
