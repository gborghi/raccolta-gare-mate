*Maximum coins on 8×8 grid: at most one face-up per row, at most one face-down per column*

> There is an $8 \times 8$ grid. On some cells, coins are placed either face-up or face-down, satisfying the following conditions:
> \begin{itemize}
> \item If a cell contains a face-up coin, then no other cell in the same row contains a face-up coin.
> \item If a cell contains a face-down coin, then no other cell in the same column contains a face-down coin.
> \end{itemize}
> What is the maximum number of coins that can be placed on the grid?
