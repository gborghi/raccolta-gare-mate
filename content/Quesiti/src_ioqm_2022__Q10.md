*10-digit number 9876543210, count new numbers obtained by swapping disjoint adjacent-digit pairs*

> Consider the 10-digit number $M = 9876543210$. We obtain a new 10-digit number from $M$ according to the following rule: we choose one or more disjoint pairs of adjacent digits in $M$ and interchange the digits in these chosen pairs, keeping the others in their places, we get $M_1 = 9876543210$. Note that any number of (disjoint) pairs can be interchanged. Find the number of new numbers that can be so obtained from $M$.
