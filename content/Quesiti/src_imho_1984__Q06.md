*Odd integers with ad=bc, a+d=2^k, b+c=2^m imply a=1*

> Let $a, b, c, d$ be odd integers such that $0 < a < b < c < d$ and $ad = bc$. Prove that if $a + d = 2^k$ and $b + c = 2^m$ for some integers $k$ and $m$, then $a = 1$.
