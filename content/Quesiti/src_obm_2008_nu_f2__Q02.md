# Olimpíada Brasileira de Matemática 2008 — Nível Universitário — Fase 2 — Quesito 2

*Finite-complement characterization via linear equation solution count*

> Let $\mathbb{N}=\{0,1,2,3,\ldots\}$. Given sets $A,B\subseteq\mathbb{N}$, for each positive integer $n$ denote by $r(A,B,n)$ the number of solutions to the equation $a+b=n$ with $a\in A$, $b\in B$.
> 
> Prove that there exists $n_0\in\mathbb{N}$ such that $r(A,B,n+1)>r(A,B,n)$ for all $n>n_0$ if and only if both $\mathbb{N}\setminus A$ and $\mathbb{N}\setminus B$ are finite.
