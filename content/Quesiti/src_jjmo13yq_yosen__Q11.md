# Japan Junior Mathematical Olympiad 2015 — Yosen (Preliminary) — Quesito 11

*Five points, four collinear, with given distances and equal angles; find BC/AD*

> In a plane there are $5$ points $A, B, C, D, O$. Among them, $A, B, C, D$ lie in this order on the same straight line, while $O$ is not on that line. It holds that $OA = 16$, $OB = 10$, $OC = 9$, $OD = 12$, and $\angle AOB = \angle COD$. Find the value of $\dfrac{BC}{AD}$. Here $XY$ denotes the length of segment $XY$.
