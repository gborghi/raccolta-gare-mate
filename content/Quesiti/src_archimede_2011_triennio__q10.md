*Numero x con uguaglianze ax+b=bx+c=cx+a*

> a, b e c sono tre numeri reali, tutti diversi tra loro. Per quanti numeri reali x, al
> massimo, possono valere le uguaglianze: ax + b = bx + c = cx + a?
>
> - **(A)** Nessuno,
> - **(B)** 1,
> - **(C)** 3,
> - **(D)** 4,
> - **(E)** almeno 5.

*Number x with equals ax+b=bx+c=cx+a*

> A, b, and c are three real numbers, all different from each other. For how many real numbers x, at most, can the equations be: ax + b = bx + c = cx + a?
>
> - **(A)** No one.
> - **(B)** 1,
> - **(C)** 3,
> - **(D)** 4,
> - **(E)** At least five.
