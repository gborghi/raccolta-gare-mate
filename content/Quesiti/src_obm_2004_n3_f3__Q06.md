*Periodic points of a planar map; prove A_b has a minimum and find it*

> Let $a$ and $b$ be real numbers. Consider the function $f_{a,b} : \mathbb{R}^2 \to \mathbb{R}^2$ defined by
> $$f_{a,b}(x, y) = (a - by - x^2,\; x).$$
> For $P = (x, y) \in \mathbb{R}^2$, define $f_{a,b}^0(P) = P$ and $f_{a,b}^{k+1}(P) = f_{a,b}\!\left(f_{a,b}^k(P)\right)$ for each non-negative integer $k$.
> 
> The set $\operatorname{per}(a, b)$ of **periodic points** of $f_{a,b}$ is the set of all points $P \in \mathbb{R}^2$ for which there exists a positive integer $n$ such that $f_{a,b}^n(P) = P$.
> 
> Fix a real number $b$. Prove that the set
> $$A_b = \{\, a \in \mathbb{R} \mid \operatorname{per}(a, b) \neq \varnothing \,\}$$
> has a minimum element. Calculate this minimum element.
