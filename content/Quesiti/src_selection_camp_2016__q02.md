*Find minimum n for set with exactly 2015 good partitions*

> Let $A$ be a set consisting of $n$ distinct positive integers. A pair of subsets $(A_1, A_2)$ is called a **good partition** if it satisfies the following conditions:
> - $A_1 \neq \emptyset$, $A_2 \neq \emptyset$.
> - $A_1 \cap A_2 = \emptyset$, $A_1 \cup A_2 = A$.
> - The least common multiple of all elements of $A_1$ equals the greatest common divisor of all elements of $A_2$.
> 
> Find the smallest value of $n$ such that there exists a set $A$ with exactly $2015$ good partitions. (Here $\emptyset$ denotes the empty set.)
