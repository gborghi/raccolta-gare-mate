# Olimpíada Brasileira de Matemática 2006 — Nível Universitário — Fase 1 — Quesito 5

*Find a particular solution of a linear ODE with zero initial data*

> The functions $y_1(t) = (1 + t^2)\,e^{t}$, $y_2(t) = (1 + t^2)\,e^{-t}$, and $y_3(t) = (-1 + t^2)\,e^{-t}$ are solutions of the differential equation
> $$y''(t) + a(t)\,y'(t) + b(t)\,y(t) = c(t),$$
> where $a(t)$, $b(t)$, $c(t)$ are twice differentiable functions.
> 
> Find a function $y(t)$ such that
> $$y''(t) + a(t)\,y'(t) + b(t)\,y(t) = c(t), \quad y(0) = 0, \quad y'(0) = 0.$$
