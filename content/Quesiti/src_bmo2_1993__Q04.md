*Average of digit-difference sums over all $m$-digit numbers with fixed digit counts*

> Let $Z(m, n)$ denote the set of all $m$-digit positive integers (in base $10$) which have precisely $a$ ones, $b$ twos, $c$ threes, $\ldots$, $n$ nines (so that $a + b + \cdots + n \le m$, with the remaining digits being zeros). For each $N \in Z(m, n)$, define $d(N)$ to be the sum of the absolute values of all pairwise differences of the consecutive digits of $N$. Find the average value of $d(N)$ as $N$ ranges over all possible elements of $Z(m, n)$.
