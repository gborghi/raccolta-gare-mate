# Coupe Animath Automne 2019 — Quesito 13

*Integer with exactly 300 ones, no zeros, and perfect square (lycée version)*

> Does there exist an integer whose decimal representation contains exactly 300 digits equal to $1$, no digit equal to $0$, and which is a perfect square?\\
> We say that an integer $n$ is a perfect square if there exists an integer $k$ such that $n = k^2$. For example, $9 = 3 \times 3$ is a perfect square but $2$ is not.
