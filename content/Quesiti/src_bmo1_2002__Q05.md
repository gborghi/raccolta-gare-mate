*Strictly increasing function on non-negative integers with functional equation, find f(2001)*

> $f$ is a function from $\mathbb{Z}^+$ to $\mathbb{Z}^+$, where $\mathbb{Z}^+$ is the set of non-negative integers, which has the following properties:
> a) $f(n+1) > f(n)$ for each $n \in \mathbb{Z}^+$,
> b) $f(f(n)) = f(n) + n$ for all $m$, $n \in \mathbb{Z}^+$.
> Find all possible values of $f(2001)$.
