*Find all functions f: R->R satisfying a functional equation.*

> Find all functions $f : \mathbb{R} \to \mathbb{R}$ which satisfy
> $$f(f(x) - y) = f(x) + f(f(y) - f(-x)) + x \quad \text{for all real } x, y.$$
