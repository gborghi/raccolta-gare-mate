*Count arrangements of letters where each letter is adjacent to an identical letter*

> Find the number of different arrangements in a row of the letters
> $$A, A, A, A, B, B, B, B, C, C, C, C$$
> such that each letter is adjacent to an identical letter.
> 
> Indicate if you can a generalisation to the case of $n$ letters each appearing five times.
