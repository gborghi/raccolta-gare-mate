*A function f on pairs of nonnegative integers satisfies given recurrences; given f(a,b)=n, find the number of integral solutions x of f(a,x)+f(b,x)=n.*

> A function $f$ from the pairs of nonnegative integers to the real numbers satisfies the following conditions:
> $$f(0,0) = 0, \qquad f(2x,2y) = f(2x+1,2y+1) = f(x,y),$$
> $$f(2x+1,2y) = f(2x,2y+1) = f(x,y) + 1$$
> for all nonnegative integers $x, y$. Let $n$ and $a, b$ be nonnegative integers such that $f(a,b) = n$. Find the number of integral solutions $x$ of the equation $f(a,x) + f(b,x) = n$.
