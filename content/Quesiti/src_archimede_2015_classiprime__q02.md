*Interpretazione di frazione a piani come divisioni annidate*

> Il valore di x è il seguente:
> x =
> A
> B
> C
> D
> .
> Allora:
> A x = A : (B : C : D)
> B x = A : (B : (C : D))
> C x = A : B : C : D
> D x = A : B : (C : D)
> E x = A : (B : C) : D
> F nessuna delle altre
> risposte è esatta
