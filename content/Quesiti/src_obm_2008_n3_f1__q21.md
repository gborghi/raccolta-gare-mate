*Count functions f(x)=cx/(2x+3) with f(f(x))=x for all x*

> Consider the function $f$, defined on the set of real numbers and satisfying $f(x) = \dfrac{cx}{2x+3}$, for all $x \neq -\dfrac{3}{2}$. Determine the number of such functions $f$ for which $f(f(x)) = x$, for all $x$ for which $f(f(x))$ is well defined.
> 
> (A) $0$ (B) $1$ (C) $2$ (D) $4$ (E) infinitely many
