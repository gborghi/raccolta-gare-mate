*Tokens in a 30x30 grid: at most one per cell, every empty cell shares row/column with a token, every token has at most one other token in its row or column; find the smallest k so that every k x k subsquare always contains a token*

> Theo places tokens in the cells of a table of size $30 \times 30$ while respecting the following rules:
> 
> $\triangleright$ Each cell contains at most one token.
> 
> $\triangleright$ For each empty cell, there is, among the cells of the same row or the same column, at least one cell containing a token.
> 
> $\triangleright$ For each token placed, there is, among the cells of the same row or the same column, at most one other cell containing a token.
> 
> Determine the smallest integer $k$ satisfying the following property: whatever the arrangement chosen by Theo, every $k \times k$ square of the grid contains at least one cell with a token.
