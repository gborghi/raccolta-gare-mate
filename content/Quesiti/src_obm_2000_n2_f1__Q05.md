# Olimpíada Brasileira de Matemática 2000 — Nível 2 — Fase 1 — Quesito 5

*Effect on class averages after transferring last student from Class A to Class B*

> The $61$ students approved in a competition, whose scores were all distinct, were distributed into two groups according to their ranking: the top $31$ were placed in Class A and the next $30$ in Class B. The averages of both classes in the competition were then calculated. Afterwards, it was decided to transfer the last-placed student of Class A to Class B. Which of the following statements is certainly true?
> 
> A) The average of Class A improved, but Class B's worsened.
> B) The average of Class A worsened, but Class B's improved.
> C) The averages of both classes improved.
> D) The averages of both classes worsened.
> E) The averages of the classes can improve or worsen, depending on the candidates' scores.
