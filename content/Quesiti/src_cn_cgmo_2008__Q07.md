*Count harmonic 2008x2008 chessboards filled with four letters*

> For a chessboard of the size $2008 \times 2008$, in each case (they all have different colors) write one of the letters $C$, $G$, $M$, $O$. If every $2 \times 2$ square contains all these four letters, we call it a "harmonic chessboard." How many different harmonic chessboards are there? (Posed by Zuming Feng)
