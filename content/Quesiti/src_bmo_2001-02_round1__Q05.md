# British Mathematical Olympiad 2001-02 — Round 1 — Quesito 5

*Find all values of f(2001) for increasing function satisfying functional equation*

> $f$ is a function from $\mathbb{Z}^+$ to $\mathbb{Z}^+$, where $\mathbb{Z}^+$ is the set of non-negative integers, which satisfies
> a) $f(n+1) > f(n)$ for each $n \in \mathbb{Z}^+$;
> b) $f(n + f(m)) = f(n) + m + 1$ for all $m, n \in \mathbb{Z}^+$.
> Find all possible values of $f(2001)$.
