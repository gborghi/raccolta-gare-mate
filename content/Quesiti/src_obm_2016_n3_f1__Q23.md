# Olimpíada Brasileira de Matemática 2016 — Nível 3 — Fase 1 — Quesito 23

*Function f summing prime exponents; comparing values and evaluating f(2016)*

> Consider the function $f$ such that if $n=p_1^{a_1}\cdots p_k^{a_k}$ is the prime factorization of $n$, then $f(n)=a_1+\dots+a_k$, with the convention used in the proof. The function satisfies: for $m>n$ with $m$ a multiple of $n$, $f(m)>f(n)$. By induction, if $\frac{n}{p}$ has $t$ prime factors then $f(\frac{n}{p})\geq t$ and $f(n)>f(\frac{n}{p})\geq t$, so $f(n)\geq t+1$. Using this, compute $f(2016)=f(2^5\cdot 3^2\cdot 7)=5+2+1+1=9$. Determine the required value (i.e. $f(2016)$ or the related bound). [Answer key value: $9$.]
