# Olimpíada Brasileira de Matemática 2005 — Nível 3 — Fase 1 — Quesito 16

*Condition for addition to distribute over multiplication: a+(bc)=(a+b)(a+c)*

> Let $a$, $b$ and $c$ be real numbers. By the distributive property of multiplication over addition, it is true that $a \times (b + c) = (a \times b) + (a \times c)$. The distributivity of addition over multiplication, that is, $a + (b \times c) = (a + b) \times (a + c)$, is not always true, but it occurs, that is, it is verified, if and only if:
> (A) $a = b = c$ or $a + \tfrac{1}{3} = 0$ (B) $a = b = c$ (C) the equality never occurs (D) $a + b + c = 1$ or $a = 0$ (E) $a = b = c = 0$
