*Prove a sum inequality for real sequences with equality condition*

> If $a_0 = a_1 = 0$, $a_2, \ldots, a_n$ are real numbers, then
> $$\sum_{i=1}^{n}(a_i - a_{i-1})^2 \geq \frac{1}{n}\left(\sum_{i=1}^{n}(a_i - a_{i-1})\right)^2,$$
> equality holding if and only if $a_{i+1} - a_i = a_i - a_{i-1}$ ($0 \leq i \leq n$).
