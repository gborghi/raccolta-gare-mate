# Japan Junior Mathematical Olympiad 2015 — Yosen (Preliminary) — Quesito 2

*Count labelings 1,2,3,4 of ten tangent circles with adjacent labels distinct*

> There are $10$ circles of equal size, arranged as shown (a triangular stack with $4$ circles in the bottom row, then $3$, then $2$, then $1$ on top; each circle is tangent to the circles it touches). We write a positive integer in each circle so that both of the following conditions hold:
> \begin{itemize}
> \item The integer $1$ is written in exactly $1$ circle, $2$ in exactly $2$ circles, $3$ in exactly $3$ circles, and $4$ in exactly $4$ circles.
> \item Any two mutually tangent circles contain different integers.
> \end{itemize}
> How many such ways are there? Arrangements obtained from one another by rotation or reflection are regarded as the same.
