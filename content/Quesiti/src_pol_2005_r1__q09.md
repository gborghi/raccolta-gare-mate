*Find all real a so that a given recurrence sequence is 8-periodic.*

> Determine all real numbers $a$ such that the sequence $(x_n)$ given by
> $$x_0 = \sqrt{3}, \quad x_{n+1} = \frac{1 + a x_n}{a - x_n} \quad \text{for } n = 0, 1, 2, \ldots$$
> satisfies the condition $x_{n+8} = x_n$ for all $n \ge 0$.
