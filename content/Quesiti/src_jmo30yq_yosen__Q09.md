*Multiplicative functional equation f(mn)=f(m)^f(n), find minimum of f(f(n))*

> Let $f$ be a function defined on the set of integers $\ge 2$ taking values in the set of integers $\ge 2$, such that for all integers $m, n \ge 2$,
> $$f(mn) = f(m)^{f(n)}.$$
> Find the minimum possible value of $f(f^{(0)})$, where $f^{(n)}$ denotes the $n$-fold composition of $f$.
> 
> *(Note: the exact expression in the exponent/composition was not fully legible in the scan. Based on context and the answer key, the problem asks for the minimum value attained by $f$ composed with itself over all valid functions $f$, which equals $4$.)*
