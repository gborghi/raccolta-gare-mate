*Find all n for which number of ordered pairs (x,y) with 1/x+1/y=1/n equals 5*

> For any positive integer $n$, let $s(n)$ denote the number of ordered pairs $(x, y)$ of positive integers for which
> $$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}.$$
> Determine all those $n$ for which $s(n) = 5$.
