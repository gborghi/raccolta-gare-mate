*Find all n for which n points (no 3 collinear) can be colored so every triangle has no obtuse angle*

> Determine all values of $n$ for which there exists a set $S$ of $n$ points, with no three collinear, having the following property: it is possible to color all points of $S$ so that every angle determined by three points of $S$ — whether all three of the same color or of three different colors — is non-obtuse. The number of available colors is unlimited.
